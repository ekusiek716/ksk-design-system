# Cake.jp 固有コンポーネント一覧

## EC・コマース系

| # | コンポーネント | 説明 |
|---|--------------|------|
| 1 | **ProductCard** | 商品カード（画像・価格・評価・ショップ名・タグ付き） |
| 2 | **ProductCTA** | 商品CTA（カートに入れる・タイムセール・予約・再入荷通知等） |
| 3 | **ProductCarousel** | 商品カルーセル（セクション見出し + レスポンシブカードサイズ） |

| 6 | **ProductOption** | 商品オプション選択（数量・セレクト・テキスト・チェックボックス・画像・ろうそく本数・レター） |
| 7 | **PriceDisplay** | 価格表示（税込・打ち消し線・セール価格） |
| 8 | **QuantitySelector** | 数量変更（+/- ボタン + ゴミ箱アイコン） |
| 9 | **CouponCard** | クーポンカード（取得ボタン付き、縦/横レイアウト） |
| 10 | **OrderSummary** | 注文合計（明細行 + CTAボタン） |
| 11 | **OrderSection** | 注文確認セクション（通常/空/エラー状態） |
| 12 | **DeliveryCalendar** | 配送日カレンダー（日付選択 + 配送可否状態） |
| 13 | **CountdownTimer** | タイムセールカウントダウン（開始前/セール中/終了状態） |
| 14 | **ShopCard** | ショップカード（商品画像・評価・配送日・ブックマーク） |
